# Next.js 프로젝트 세팅

## 1. Node 설치

[Node 공식 홈페이지](https://nodejs.org/ko)에서 Node 다운로드 받아 설치

## 2. 프로젝트 다운로드

```bash
git clone https://github.com/project-name.git
```

## 3. 패키지 설치

yarn 패키지매니저 설치

```bash
npm install -g yarn
```

패키지 설치

```bash
yarn install
```

## 4. 프로젝트 실행

```bash
yarn dev
```
